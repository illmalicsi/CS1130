Project 2: Grade Computation

The Grade Calculator and Converter is a tool designed to calculate and convert grades based on user input. Users input three values: Score (the total points earned), Items (the total points possible), and Base (the minimum possible score). The tool then calculates the grade using the formula Math.round(Score/Items)*(100-Base)+Base, which calculates the percentage score, scales it to a range based on the Base value, and rounds to the nearest whole number.

Finally, the numerical grade is converted into a literal grade using a predefined scale. This tool serves as a valuable resource for educators and students, providing a quick and easy way to calculate and understand grades based on a variety of scoring systems.

Project 3: Income Tax Calculator

A user-friendly tool designed to streamline the process of computing income tax based on a specified table. This project eliminates the need for prompts by utilizing HTML input tags, allowing users to input their taxable income directly. The calculation follows a straightforward table that considers different income brackets and corresponding rates. Users can easily determine their income tax liability by inputting their net taxable income, and the system will automatically compute the Basic Tax and applicable Rate, providing a clear breakdown for better financial planning. Say goodbye to complex calculations and welcome a simplified, efficient solution for income tax computation.
